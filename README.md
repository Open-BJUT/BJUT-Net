## 项目简介

北京工业大学（BJUT）校园网一键认证APP是一个开源项目，旨在使北京工业大学学生和员工能够更方便快捷地连接校园网。

该**安卓**应用程序集成了一键认证功能，用户只需登录一次并保存登录信息，即可使用应用程序自动登录并连接学校的校园网。这样，用户就可以省去繁琐的输入账户和密码的过程，毫不费力地连接到网络。
